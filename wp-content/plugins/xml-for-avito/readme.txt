== Description ==

Создаёт XML-feed для загрузки на Авито. Для работы требуется WooCommerce!

PRO версия: [https://icopydoc.ru/product/xml-for-avito-pro/](https://icopydoc.ru/product/xml-for-avito-pro/?utm_source=wp-repository&utm_medium=organic&utm_campaign=xml-for-avito&utm_content=readme&utm_term=pro-version)

== Installation ==

В этом разделе описывается, как установить плагин и заставить его работать.

1. Загрузите папку `xml-for-avito` в `/wp-content/plugins/`
1. Активируйте плагин на странице ‘Плагины’ WordPress
1. Перейдите на страницу Экспорт Avito чтобы настроить плагин

== Frequently Asked Questions ==

= Какие плагины интернет-магазинов поддерживает Ваш плагин? =

Только WooCommerce.

= Как создать XML фид? =

Подробная инструкция со скриншотами тут
Перейдите в меню Экспорт Avito. В поле «Автоматическое создание файла» выберите любое значение (кроме «Отключено»). Обязательно нажмите «Сохранить» внизу страницы.
Через 1 — 7 минут (зависит от числа товаров), фид будет сгенерирован и вместо данного сообщения появится ссылка.
